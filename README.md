# Dogfight

Arcade 3D dogfight combat flight simulator built with Three.js. Engage AI fighters, dodge missiles, and fly through canyons.

## Controls

| Key | Action |
|---|---|
| Mouse | Aim (War Thunder-style instructor) |
| Left Click | Machine gun |
| Right Click (hold) | Zoom |
| Space | Fire missile (requires lock) |
| F | Deploy flares |
| Scroll Wheel | Throttle |
| Ctrl (at 0% throttle) | Airbrake |
| V | Toggle camera |

## Features

- Proportional navigation missiles with smoke trails
- Flare countermeasures and jink evasion
- Sonic boom effects when breaking Mach 1
- Wingtip vortex contrails during high-G turns
- Terrain with mountains and canyons
- Ammo balloons scattered across the map
- Kill rewards: full health + ammo replenish

## Run

```bash
npm install
npm run dev
```
